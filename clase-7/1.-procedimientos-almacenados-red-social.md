## Procedimiento Almacenado simple

```sql
-- PROCEDIMIENTO
CREATE PROCEDURE sp_CrearUsuario
    @NombreUsuario VARCHAR(50),
    @Email VARCHAR(100),
    @NuevoUsuarioID INT OUTPUT
AS
BEGIN
    SET NOCOUNT ON;
    INSERT INTO Usuarios (NombreUsuario, Email, FechaRegistro)
    VALUES (@NombreUsuario, @Email, GETDATE());
    SET @NuevoUsuarioID = SCOPE_IDENTITY();
END;

```

## Ejecutar el Procedimiento

```sql
DECLARE @NuevoID INT;
EXEC dbo.sp_CrearUsuario 'sofia', 'sofia@correo.com', @NuevoID OUTPUT;
SELECT @NuevoID AS Nuevo_id_User;
```
